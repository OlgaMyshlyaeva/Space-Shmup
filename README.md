# Space-Shmup
A fast-paced top-down space shooter. Battle waves of enemies and upgrade your ship's weapons.

## Key Features
* **Scrolling Background:** Seamless starfield parallax effect.
* **Weapon System:** Dynamic projectile spawning and power-up collection.
* **Enemy AI:** Diverse movement patterns and destruction logic.
* **Collision Matrix:** Fine-tuned physics interaction between player, enemies, and shields.

## Tech Stack
* **Engine:** Unity
* **Language:** C#
